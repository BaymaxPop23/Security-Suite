# APKSlayer - Requirements
#
# Core dependencies: None (stdlib only)
#
# External tools required:
#   - jadx: APK decompilation (https://github.com/skylot/jadx)
#   - adb: Android Debug Bridge (Android SDK)
#
# Optional dependencies for PDF reports:
# weasyprint>=52.0    # PDF generation (uncomment if needed)
# wkhtmltopdf         # Alternative PDF generation

# Optional dependencies for enhanced dynamic analysis:
# uiautomator2>=3.0.0   # UI automation (recommended)
# frida-tools>=12.0.0   # Runtime instrumentation (requires frida-server on device)
# mitmproxy>=10.0.0     # Traffic interception and analysis

# To install all optional dynamic analysis dependencies:
# pip install uiautomator2 frida-tools mitmproxy
